# NCCC Supplementary Material

## Second Task: Standing Orders

The following represents a complete, fully-implemented Java project, which can be used as a good educational reference. The purpose of this project is to demonstrate the possibility of mimicking the basic functionality of a Standing Order System. In addition, the project is written in such a manner that demonstrates the outcome as tested Use Cases.

The general aims are:

* To solve a problem in an Object-Oriented manner.
* To demonstrate file IO (input and output) in Java.
* To demonstrate the use of Abstraction and Inheritance in Java.
* To demonstrate the benefits of Exception Handling in Java.
